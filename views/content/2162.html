The Ontario Building Code | Outdoor Design Temperatures

9.33.3.2. door Design temperatures

(1) The door conditions to be used in designing heating, ventilating and air-conditioning systems shall be the appropriate values for the location as set out in MMAH Supplementary Standard SB-1, "Climatic and Seismic Data", using 2.5 percent design temperature criteria.
