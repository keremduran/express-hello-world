7.4.10.2. Hydraulic Loads for Fixtures

(1) The hydraulic load from a fixture that is listed in Table 7.4.9.3. is the number of fixture unitsset forth in the Table.

(2) Except as provided in Sentence (1), the hydraulic load from a fixture that is not listed in Table 7.4.9.3. is the number of fixture units set forth in Table 7.4.10.2. for the trap of the sizethat serves the fixture.
